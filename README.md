# TinyShell

### 使用方法

1. dev c++下运行

2. windows 下使用 makefile

   - win+R输入cmd打开命令行窗口

   - 使用命令 cd 进入TinyShell项目所在的本地磁盘（这里举例为e盘）

     ```
     cd :d /e
     ```

   - 使用命令 cd 进入TinyShell项目所在的文件夹，例如：

     ```
     cd 大二秋（2021-2022）\程序设计基础\大作业\组队部分\TinyShell
     ```

   - 输入命令对代码进行编译

     ```
     make clean
     make
     ```

   - 双击当前目录下的main.exe即可运行
